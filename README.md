## Descripción del proyecto

Este proyecto desarrolla un sistema de clasificación supervisada para predecir el nivel de riesgo de **burnout académico** en estudiantes universitarios a partir de variables relacionadas con el uso de herramientas de Inteligencia Artificial, desempeño académico y hábitos de estudio.

El objetivo principal es comparar el desempeño de diferentes algoritmos de aprendizaje automático para identificar cuál ofrece la mejor capacidad de predicción sobre tres niveles de riesgo de burnout:

- Bajo (Low)
- Medio (Medium)
- Alto (High)

Para ello, se realizó un proceso completo de ciencia de datos que incluye la limpieza y transformación de los datos, análisis exploratorio (EDA), ingeniería de características, entrenamiento de modelos, optimización de hiperparámetros mediante validación cruzada y evaluación utilizando métricas de clasificación multiclase.

## Objetivos

- Realizar el preprocesamiento y limpieza del conjunto de datos.
- Analizar la distribución de las variables mediante técnicas de análisis exploratorio.
- Entrenar diferentes modelos de clasificación.
- Optimizar los hiperparámetros utilizando GridSearchCV y RandomizedSearchCV.
- Comparar el desempeño de los modelos utilizando Accuracy, Precision, Recall y F1-Score Macro.
- Analizar la importancia de las variables que más influyen en la predicción del riesgo de burnout.

## Modelos implementados

- Regresión Logística
- Máquina de Soporte Vectorial (SVM)
- Random Forest
